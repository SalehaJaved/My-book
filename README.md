# Physical AI & Humanoid Robotics 

This repository contains the source code and content of the book **"Physical AI & Humanoid Robotics: Embodied Intelligence in the Physical World"**.

The book explores how artificial intelligence moves beyond screens into the physical world through humanoid robots, sensors, actuators, and learning systems.

## Topics Covered
- Physical AI & Embodied Intelligence
- Humanoid Robot Design
- Perception–Action Loops
- Robot Learning & Adaptation
- Human–Robot Interaction
- Real-World AI Deployment

This project is intended for students, educators, and researchers working in AI, robotics, and intelligent systems.
